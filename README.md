# Personal Portfolio

### 🔴 [Live Demo: Click Here to View Portfolio](https://iamfaizan-ok.github.io/my-portfolio/)

A professional and responsive personal portfolio website showcasing my skills, projects, and experience as a Full-Stack Developer & Backend Specialist. 

## Features
- **Responsive Design**: Looks great on both desktop and mobile devices.
- **Modern UI**: Clean and minimal aesthetics with smooth transitions.
- **Sections**: About Me, Projects, Skills, and Contact.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla)
- FontAwesome (Icons)
- Google Fonts (Outfit)

## Setup & Run Locally
1. Clone the repository or download the ZIP file.
2. Open the `index.html` file in any modern web browser.
3. Enjoy exploring the portfolio!
